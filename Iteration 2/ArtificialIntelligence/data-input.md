## Data input

### Auto-complete
Here is a list of all the suggestion words used in the project:
* as + \[soon as possible\]
* even + \[if\]
* by \[the time\]
* case + \[of\]
* followed + \[by\]
* assuming + \[that\]
* so + \[that\]
* other + \[words\]
* matter + \[how\]

### Text analysis
Here is a list of all the keywords collected from the text:
* tomorrow
* today
* yesterday
* at \[hh\](:\[mm\])
* at \[location\]
* in \[d\] days
* in \[d\] weeks
* in \[location\]
* (in the) next \[d\] days
* (in the) next \[d\] weeks
* next week
* next day
